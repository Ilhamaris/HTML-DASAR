My First Heading

HTML (Hypertext Markup Language) adalah bahasa markah standar yang digunakan untuk membuat struktur halaman web. Dengan HTML, pengembang web dapat menandai elemen-elemen seperti teks, gambar, tautan, dan bagian lain dari halaman web. Ini memberikan dasar untuk menampilkan konten secara visual di peramban web. HTML menggunakan serangkaian tag atau elemen untuk mengatur dan memformat konten sehingga dapat ditampilkan dengan benar. Sebagai bahasa markup, HTML digunakan untuk menafsirkan dan menyusun konten web.
